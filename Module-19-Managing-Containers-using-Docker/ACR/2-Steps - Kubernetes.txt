Deploying Kubernetes

1. Create Resource - Kubernetes Service: Node Count:1, Authentication Method: Service Principal
2. Wait 10 min
3. Understand how to give access for Azure Kubernetes to the Azure Container Registry - aks_permissions_commands.txt
4. Open - aks_permissions_commands.txt
5. Open Cloud Sheel and execute commands from aks_permissions_commands.txt
6. Create AKS yaml files for deloyment. Refer - aks_app.yaml
7. Open resource AKS Cluster\Workloads\Add
8. Paste the aks_app.yaml file content
9. Open resource AKS Cluster\Services and Ingress\Add
10. Paste the aks_service.yaml file content
11. Open the External IP address generated by the service
